# 1Thing
### One Task at a Time. One Life.

> *"The man who chases two rabbits catches neither."* — Confucius

---

## What is this?

**1Thing** is a free, open-source productivity app built around a single radical idea: you only get **one task at a time**.

Most apps overwhelm you with lists, tags, priorities, reminders, and integrations. 1Thing does the opposite: it forces you to name the one thing that matters right now — then holds you to it.

---

## How to run it

1. Download Source code (zip) file from the Releases tab or visit [our website](https://patatasfritasofficial.github.io/1Thing).
2. Extract the zip and open `index.html` in any modern web browser.
3. That's it. No installation. No sign-up. No account. No internet required after first load.

Optional: Bookmark the HTML file or our website for easier access ;)

---

## How to use it

1. **Write your one thing** — the single task that matters right now
2. **Pick a timer** — Pomodoro, Stopwatch, Custom (hours / minutes / seconds), or none
3. **Review & commit** — confirm on the preview screen
4. **Click "Start now"** — the timer begins, you're locked in
5. **Do the thing**
6. **Click "✓ Done — I did it"** — watch your streak grow
7. **Start your next task** or **End this day**

---

## Features

- **One task at a time** — no lists, no overwhelm, pure focus
- **Flexible timers** — Pomodoro (25 min), Stopwatch, fully custom (h/m/s), or no timer
- **Custom timer gate** — with a custom timer, done is only available once time is up
- **Task preview screen** — review before you commit; cancel without penalty
- **Live 24-hour clock** — day, date, and month on the home screen
- **Day progress bar** — a quiet reminder that time is passing
- **Streak tracking** — consecutive days you finished your thing
- **Archive** — a clear log of every completed and abandoned task
- **Motivational quotes** — random real quotes shown on day completion
- **Confetti** — because finishing deserves a celebration
- **Backup & restore** — export history and streak as `.json`, restore anytime
- **100% local** — all data stays in your browser, nothing leaves your device

---

## Data & privacy

All data is stored in `localStorage`. No servers. No accounts. No tracking.

**Backup:** open Archive → Export backup  
**Restore:** open Archive → Restore backup  
**Clear:** open Archive → Clear all history
